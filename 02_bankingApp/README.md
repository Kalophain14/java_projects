## What I Learned
Practiced core Java fundamentals:

- Variables: primitive types (int, double, boolean) and class-level fields
- Variable scope: local variables inside methods vs class-level fields
- Method creation: breaking logic into focused private helper methods
- Access modifiers: private methods/fields and public entry points
- Scanner: reading and validating user input from the console
- printf formatting: displaying currency values with %.2f
- Conditionals: if/else for input validation and switch-case for menu navigation"
