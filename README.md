## Calculator

An Apple iOS styled calculator built with Java Swing.

### Features

- Basic arithmetic: `+`, `-`, `Ã—`, `Ã·`
- Percentage and square root (`âˆš`)
- Sign toggle (`+/-`)
- Keyboard support
- Apple-inspired dark UI

### Keyboard Shortcuts

| Key             | Action            |
| --------------- | ----------------- |
| `0-9`           | Number input      |
| `+` `-` `*` `/` | Operators         |
| `=` or `Enter`  | Calculate         |
| `.`             | Decimal           |
| `%`             | Percentage        |
| `Backspace`     | Delete last digit |
| `Escape`        | Clear (AC)        |

### Running the Project

1. Open `Calculator` folder in IntelliJ IDEA
2. Run `Main.java`
