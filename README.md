# Java Calculator

A Java Swing calculator styled after the Apple iOS calculator.

## Features

- Basic arithmetic: `+`, `-`, `×`, `÷`
- Percentage and square root (`√`)
- Sign toggle (`+/-`)
- Keyboard support
- Apple-inspired dark UI

## Keyboard Shortcuts

| Key             | Action            |
| --------------- | ----------------- |
| `0-9`           | Number input      |
| `+` `-` `*` `/` | Operators         |
| `=` or `Enter`  | Calculate         |
| `.`             | Decimal           |
| `%`             | Percentage        |
| `Backspace`     | Delete last digit |
| `Escape`        | Clear (AC)        |

## Running the Project

1. Open in IntelliJ IDEA
2. Run `Main.java`
